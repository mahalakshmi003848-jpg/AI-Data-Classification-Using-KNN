# AI Data Classification Using KNN

## Project Overview

This project implements a Machine Learning Classification Model using the Iris Dataset and the K-Nearest Neighbors (KNN) algorithm.

## Technologies Used

* Python
* Scikit-Learn
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Workflow

1. Load Iris Dataset
2. Feature Scaling using StandardScaler
3. Train-Test Split
4. KNN Model Training
5. Prediction
6. Performance Evaluation

## Evaluation Metrics

* Accuracy Score
* F1 Score
* Confusion Matrix
* Classification Report
